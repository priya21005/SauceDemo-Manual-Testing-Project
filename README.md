🧪 SauceDemo Manual Testing Project
📖 Project Overview

The project demonstrates my understanding of the Software Testing Life Cycle (STLC) by creating test scenarios, designing test cases, executing tests, and documenting the results.

The objective of this project is to validate the core functionalities of the application and ensure that they meet the expected business requirements.

🌐 Application Under Test
Application Name: SauceDemo
Application Type: E-commerce Web Application
Application URL: https://www.saucedemo.com/

🎯 Project Objectives
Verify the functionality of the SauceDemo application.
Create comprehensive test scenarios and test cases.
Validate positive and negative user flows.
Identify and report defects.
Practice industry-standard QA documentation.

🧩 Modules Covered
